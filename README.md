# Dynamic Pricing Regression Model

A supervised machine learning regression project designed to predict bike rental costs for dynamic pricing optimization.

The project benchmarks Ridge Regression and Random Forest Regression models to evaluate performance, generalization, and computational efficiency for real-world pricing strategy scenarios.

---

## Features

- End-to-end regression pipeline for pricing prediction
- Data cleaning and preprocessing (missing value handling, categorical encoding)
- Feature engineering with One-Hot Encoding
- Model implementation:
  - Ridge Regression
  - Random Forest Regression
- K-Fold Cross-Validation for performance stability
- Model evaluation using:
  - RÂ² Score
  - Mean Squared Error (MSE)
  - Mean Absolute Error (MAE)
- Hyperparameter optimization using GridSearch
- Performance comparison based on accuracy and training efficiency
